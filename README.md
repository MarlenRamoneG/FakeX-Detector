# FakeX-Detector
Script para detectar cuentas falsas en X usando Python y análisis de actividad de imagen.

Para comenzar, instala las dependencias necesarias y ejecuta `detector.py` con tus imágenes de entrada:

```bash
python detector.py path/a/imagenes/
```

El script actualmente contiene una función de detección de cuentas falsas como marcador de posición y se debe completar con la lógica real.
